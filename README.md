# 구글 클라우드 일일 전략 블로그

구글 클라우드 블로그 수집 결과를 바탕으로 생성한 한국어 전략형 HTML 블로그 저장소입니다.

- 최신 HTML: `docs/index.html`
- 날짜별 HTML: `docs/20260620/index.html`
- 원본 작업본: `src/20260620/index.html`
- 로컬 standalone: `/opt/data/out/gcp-daily-blog-20260620.html`

## 출처
- https://cloud.google.com/blog/products/identity-security/choice-compliance-and-collaboration-europes-path-to-open-digital-sovereignty

## 실행
정적 파일이므로 `python3 -m http.server`로 미리보기할 수 있습니다.
