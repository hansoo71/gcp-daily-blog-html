# 이미지 프롬프트 기록

## 이미지 프롬프트 1
- 목적: 대표 이미지
- 삽입 위치: 상단 도입부
- 이미지 생성 프롬프트: 한국 기업 임원을 위한 전략 블로그 히어로 이미지. 주제: 생성형 이미지 모델 대중화와 데이터 파이프라인 인공지능화가 만나는 구글 클라우드 전략. 추상적인 클라우드 데이터 흐름, 인공지능 캔버스, 보안 거버넌스 레이어를 세련된 편집 일러스트 스타일로 표현. 실제 회사 로고 없음. 이미지 안의 글자는 최소화하거나 사용하지 않음.
- 권장 비율: 가로형
- 대체 텍스트: 생성형 시각 모델과 클라우드 데이터 흐름을 표현한 추상 편집 이미지

## 이미지 프롬프트 2
- 목적: 데이터 파이프라인 설명
- 삽입 위치: 기술 변화의 의미 섹션
- 이미지 생성 프롬프트: 데이터플로우 기반 인공지능 데이터 파이프라인 인포그래픽. 스트리밍 데이터, 배치 데이터, 품질 검증, 모델 입력, 운영 모니터링이 하나의 흐름으로 이어지는 모습을 한국어 블로그에 어울리는 선명한 시각 스타일로 표현. 실제 로고 없음. 텍스트는 매우 적게.
- 권장 비율: 가로형
- 대체 텍스트: 데이터 파이프라인이 인공지능 운영으로 이어지는 과정을 표현한 인포그래픽

## 이미지 프롬프트 3
- 목적: 기업 실행 로드맵
- 삽입 위치: 실행 전략 섹션
- 이미지 생성 프롬프트: 기업 적용 로드맵 이미지. 1단계 실험, 2단계 데이터 통합, 3단계 거버넌스, 4단계 운영 확산을 상징하는 네 개의 연결된 카드와 클라우드/인공지능/보안 아이콘. 한국어 제목은 넣지 말고 시각 요소 중심. 실제 로고 없음.
- 권장 비율: 가로형
- 대체 텍스트: 기업 적용 로드맵과 거버넌스 단계를 표현한 시각 이미지
