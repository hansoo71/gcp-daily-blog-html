LLM 분석의 단가 문제 해결

Proxy model 접근은 모든 행마다 LLM을 호출하지 않고 임베딩/대체 모델을 활용해 속도와 비용을 동시에 낮춥니다. 이는 대규모 VOC, 리뷰, 콜센터, 로그 분석에서 AI SQL을 실제 운영 예산 안으로 끌어들이는 신호입니다.
